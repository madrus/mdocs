## SmartyPants syntax

Adding 

```yaml
  - smarty:
      smart_angled_quotes: true
```

to `markdown_extentions` give the possibility to print out ASCII dashes, quotes and ellipes:

You write:

```none
'single quotes'
"double qoutes"
<<angle quotes>>
... ellipsis
-- ndash
--- mdash
```

You get:

'single quotes'
"double qoutes"
<<angle quotes>>
... ellipsis
-- ndash
--- mdash
